Q-Windows are masterpieces in Security. Various technical refinements will stop any burglary attempt and protect you from unwelcome visitors. The fitting groove, for example, is positioned on the internal side of the sash, which makes it particularly difficult to force the window open.

Security Window Components

Components which distinguish a security window from a standard window:

  • Multi-Chamber frame and burglar proof security glass
  • The technology of the Multi-Chamber System ensures maximum stability
  • Window hardware fixed with high grade stainless steel screws
  • Minimum 2 internal locking points on window hinge side
  • Window Installed with hardened stainless steel bolts, amount of fasteners calculated per security level
  • Completely foamed in with high grade Polyurethane Foam for added security

Specification

Security Level One

Security Level One

Ideal for private homes, condominium units, businesses and vacation homes.

  • All of the above described security components for the frame
  • Minimum 2 internal security locks
  • Two layer 6.38 mm laminated security Glass

Security Level Two

Security Level Two

Ideal for homes of people with a high security demand. Especially for Single and Linked houses.

  • Same frame as Level One
  • Minimum 4 internal security locks
  • Two layer 10.38 mm laminated security Glass

Security Level Three

Security Level Three

Ideal for people living in exposed locations. Recommended for buildings such as bungalows, computer labs, energy centers and clinics.

  • Same frame as Level One
  • Minimum 6 internal security locks connected by hidden corner drives
  • Two layer 10.38 mm laminated security Glass

High Security Level One

High Security Level One

Special designed for high profile individuals. Recommended for villas, government offices, art galleries, museums, jewelry stores, banks and any other situation where heightened protection is needed.

  • Steel reinforced aluminium frames
  • Minimum 6 internal security locks connected by hidden corner drives
  • Four layer solid 20.28 mm laminated security Glass
  • Lockable window handle
  • Drill protection

High Security Level Two

High Security Level Two

Consists all materials and specification like High Security Level I. Ideal where Bullet Resistant windows are needed.

  • Steel covered aluminium frames
  • Bullet proofed laminated security Glass in according the demands
